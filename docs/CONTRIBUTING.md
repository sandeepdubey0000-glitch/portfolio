# Contributing

Thanks for checking out this portfolio. This repository is a demo of supply chain analytics projects and uses synthetic datasets for demonstration purposes.

If you'd like to reproduce or extend the projects locally:
1. Clone the repository.
2. Open the relevant project folder (e.g., Mytheresa-Inventory-Optimization).
3. Load the CSV into Power BI or a Python notebook.
4. Follow the README within each project for step-by-step instructions.

If you fork or adapt any material, please add attribution back to this repo and keep the LICENSE intact.
